# Cluster Snapshot Evidence

Generated by:

```bash
bash bin/debug/capture-k8s-snapshots.sh [NAMESPACE] [OUT_DIR]
```

Each run emits a UTC timestamp prefix:

- `<timestamp>-kubectl-context.txt`
- `<timestamp>-namespaces.txt`
- `<timestamp>-nodes.txt`
- `<timestamp>-pods.txt`
- `<timestamp>-services.txt`
- `<timestamp>-ingress.txt`
- `<timestamp>-pvc.txt`
- `<timestamp>-deployments.txt`
- `<timestamp>-statefulsets.txt`
- `<timestamp>-jobs.txt`
- `<timestamp>-events.txt`
- `<timestamp>-describe-ingress.txt`

Use these as reproducible operational evidence artifacts in PRs, release notes, or incident timelines.
